# DSRTC
Starting point for common library supporting DS3232 and DS3234 RTC chips. These are I2C and SPI variants of the common core which contains an accurate temperature compensated oscillator rated at Â±2 minutes per year, programmable square wave output, fixed 32kHz output, and 2 user programmable alarms which can match dates, day of week, hours, minutes, or seconds.
